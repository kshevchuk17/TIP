Динамические стили. Динамическое позиционирование

В динамическом HTML браузера MS Internet Explorer динамические стили имеют намного больше возможностей за счет следующих особенностей объектной модели: определения и переопределения таблицы стилей во время загрузки и анализа документа. Например, можно написать код условия, который загружает различные таблицы стилей в зависимости от размера экрана в ходе загрузки документа; модификации таблиц стилей документа из кода программы без его перезагрузки за счет предоставления свойства style для всех элементов документа; динамического позиционирования для доступа и манипулирования положением элементов в документе. За счет введения второй возможности динамические стили, также как и динамические тэги обладают свойствами, методами и событиями. Кроме этого, документ содержит набор таблиц стилей, связанных с ним посредством коллекции styleSheets. Свойства динамических стилей в объектной модели представлены с использованием следующих соглашений об именовании. Как вы уже знаете, в обычных CSS большинство свойств разделяют ключевые слова при помощи дефиса (-). Однако дефис интерпретируется как оператор в большинстве языков программирования и не может быть частью имени свойства CSS для представления его в объектной модели. Поэтому правило написания свойств для чувствительных к регистру языков (таких как JavaScript) определили следующим образом: первая часть правила набирается в нижнем регистре, а первый символ части после дефиса набирается прописными буквами. Если используется правило из одного слова (без дефиса), то первый символ также набирается прописными буквами. Например, правило для размещения текста по горизонтали, определяемое в каскадной таблице стилей как text-align, при использовании его в программе определяется под именем textAlign, а background будет определяться под именем Background. Имеется несколько исключений из этого правила для предотвращения совпадения с другими ключевыми словами языков программирования, когда впереди такого слова пишут префикс style. Например, правило float в CSS определяет выравнивание элемента по левому или правому краю при обтекании текстом изображения, но поскольку float является распространённым ключевым словом для определения типа данных во многих языках программирования, то правило float таблицы стилей определяется в DOM как свойство styleFloat. Возможности использования динамических стилей очень обширны.
