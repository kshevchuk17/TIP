Возможности DHTML. Примеры динамических эффектов

Следует знать, что DHTML является кроссплатформенным - независимым от платформы (аппаратных средств и операционной системы) и языка программирования. То есть обращение к свойствам и методам объектов в рамках DOM должно обеспечиваться различными операционными системами, браузерами и языками программирования, например, такими как: JavaScript, Jscript, VBScript, Java, Delphi, С++ и другими. Более того, можно комбинировать языки, вызывая, например, подпрограммы на VBScript из программ Jscript. Здесь мы будем использовать лишь язык Javascript (Jscript) и средства DHTML, связанные с этим языком, а также ОС Windows 2000/XP. Рассмотрим далее, возможности DHTML, обращая основное внимание на те из них, которые могут использоваться в приложениях экологического характера, а затем продемонстрируем некоторые из них на примерах. Динамический HTML понятие собирательное и представляет собой комбинацию некоторых новых тэгов HTML и возможностей программирования, что позволяет сделать веб - страницы интерактивными и размещать на них анимационные объекты.Благодаря DHTML страницы начинают функционировать подобно любой компьютерной программе.Динамический HTML объединяет следующие средства: язык разметки гипертекста(HTML), дополненный некоторыми дескрипторами; каскадные таблицы стилей(CSS) и послойное представление информации; язык JavaScript(или другой современный язык). Связующим звеном для перечисленных средств служит объектная модель документа(DOM). DOM уже рассматривалась ранее в лекциях.Здесь же коротко выделим основное. Любой элемент страницы(раздел или секция, заголовок, параграф, изображение, список, форма и т.п., то есть блок, заключенный в парные тэги) считается объектом.Каждому такому объекту можно присвоить имя и атрибуты(идентификаторы стиля, цвет текста и т.п.) и затем обратиться по этому имени к объекту и его атрибутам в сценарии, содержащемся в коде данной страницы.Элементы страницы затем можно будет изменять в результате определенного события(при наведении указателя мыши, щелчке, нажатии клавиши на клавиатуре, либо по истечении заданного промежутка времени).Изменяться может не только стиль или цвет текста, но и весь объект, включая код HTML, текст или рисунок. Кроме этого, имеется возможность динамически управлять стилевым оформлением страницы.В разделе 2, посвященном HTML, вы узнали, что каскадные таблицы стилей используются для управления форматированием и расположением элементов страницы.С помощью CSS описания признаков форматирования и свойств компоновки страницы можно сосредоточить в одном месте и избежать их повторного ввода в каждом отдельном дескрипторе.Еще одно преимущество CSS состоит в возможности обращаться к его правилам форматирования на языке JavaScript и манипулировать этими правилами, позволяя автоматизировать создание дизайна страниц и оснащать страницу разнообразными динамическими эффектами. Слои составляют часть технологии CSS. С помощью слоев можно описывать элементы HTML-документа, которые могут перекрываться, перемещаться, отображаться или, наоборот, скрываться, как отдельные элементы. Один из способов определения CSS-правил для слоя – использование HTML-контейнера div. Слои можно накладывать один на другой, задавая соответствующим образом их координаты. В случае наложения одного слоя на другой по умолчанию на переднем плане отображается тот, что был описан последним. Но для тэга div предусмотрен атрибут z-index, меняя значения которого можно управлять порядком наложения слоёв. Верхним окажется слой, у которого атрибут z-index имеет наибольшее значение.
